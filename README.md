# Alka-Seltzer-Report

Performing an experiment to determine factors associated with tablets disolving.

QUESTION 1: Which variables (liquid type, liquid temperature, tablet brand, and surface area) are significant in predicting dissolve time of antacid tablets?

QUESTION 2: Which combination of factors dissolves the tablets the fastest?

QUESTION 3: What is the prediction interval for the combination of factors with the fastest dissolve time and do the trial runs support the prediction?

Uses statistical analysis to answer the above questions.
